# Table JS

A customizable, feature-rich React table component built with Material-UI, supporting sorting, filtering, pagination, and advanced features like sticky columns, tooltips, and action menus.

## Features
- **Sorting**: Click column headers to sort data.
- **Filtering**: Searchable columns with text inputs.
- **Pagination**: Configurable rows per page and page navigation.
- **Sticky Columns**: Fixed columns for scrolling tables.
- **Action Menus**: Customizable actions (e.g., Edit, Share) with tooltips.
- **TypeScript Support**: Fully typed with TypeScript definitions.
- **Material-UI Styling**: Consistent, customizable styles.
